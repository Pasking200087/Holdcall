# Сборка и установка

## Требования
- Python 3.11+ (64-bit)
- pip

## Установка зависимостей
```
pip install -r requirements.txt
```

## Запуск без сборки (для разработки)
```
python main.py
```

## Сборка .exe
```
pyinstaller build.spec
```
Готовый файл: `dist/baza.exe`

## Установка на NAS / сетевой диск

1. Скопировать `dist/baza.exe` в папку на сетевом диске, например `\\NAS\shared\baza\`
2. При первом запуске автоматически создадутся:
   - `baza.db` — база данных (зашифрованные контакты)
   - `baza.key` — ключ шифрования (скрытый файл)
3. Пользователи запускают `baza.exe` напрямую с сетевого диска

## Первый вход
- Логин: `owner`
- Пароль: `owner`
- **Сразу сменить пароль!** (Аккаунт → Сменить пароль)

## Важно про безопасность
- Файл `baza.key` — не удалять и не перемещать! Без него данные нечитаемы.
- Сделать бэкап обоих файлов (`baza.db` + `baza.key`) в разные места.
- Ограничить доступ к папке через права NAS (только нужные пользователи).
- Файл `baza.db` без `baza.key` содержит только зашифрованные данные — открыть через DB Browser бесполезно.
